A pulsar is a rapidly rotating neutron star that beams electromagnetic radiation towards us, which is how we can see them. Typical rotation periods range from a millisecond to a few seconds. So each time the pulsar rotates, we observe a blip when the radiation beams towards us. Since these objects are additionally very stable rotators, they are essentially very accurate clocks with which we may make astronomical measurements. So what’s the Hulse-Taylor binary pulsar?
